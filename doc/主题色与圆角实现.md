# 主题色与自定义圆角实现

本文档介绍设置面板中「主题色」和「自定义圆角」功能的完整实现原理。

---

## 核心技术：CSS 自定义属性（CSS Variables）

两个功能都基于同一个核心技术 —— **CSS 自定义属性**（也叫 CSS 变量）。

```css
/* 定义变量 */
:root {
  --theme-color: #5d87ff;
}

/* 使用变量 */
.button {
  background: var(--theme-color);
}
```

关键特性：

- **继承性**：子元素自动继承父元素的 CSS 变量
- **可动态修改**：通过 JavaScript 的 `setProperty` 实时修改，页面立即响应
- **优先级**：inline style（`element.style`）优先级高于样式表中的选择器

```js
// 通过 JS 修改变量，页面上所有用到该变量的地方都会立即更新
document.body.style.setProperty('--theme-color', '#FF0000')
```

## 一、主题色

### 1.1 整体流程

```
用户点击色块
  → settings panel 调用 settingStore.setThemeColor(color)
    → store 更新 reactive 状态（持久化到 localStorage）
    → 调用 setArcoThemeColor(color)
      → 在 document.body.style 上设置多个 CSS 变量
        → 页面上所有引用这些变量的元素自动更新颜色
```

### 1.2 配置定义

预设颜色列表定义在 `src/config/index.ts`：

```ts
systemMainColor: [
  '#5D87FF', // 蓝色（默认）
  '#B48DF3', // 紫色
  '#1D84FF', // 浅蓝
  '#60C041', // 绿色
  '#38C0FC', // 天蓝
  '#F9901F', // 橙色
  '#FF80C8', // 粉色
] as const
```

默认值在 `src/config/setting.ts`：

```ts
systemThemeColor: AppConfig.systemMainColor[0] // '#5D87FF'
```

### 1.3 Store 层

`src/store/modules/setting.ts`：

```ts
// 状态
const systemThemeColor = ref(SETTING_DEFAULT_CONFIG.systemThemeColor)

// 设置方法
const setThemeColor = (theme: string) => {
  systemThemeColor.value = theme // 更新 reactive 状态（会自动持久化）
  setArcoThemeColor(theme) // 应用到 DOM
}
```

Store 使用 `pinia-plugin-persistedstate`，`systemThemeColor` 会自动保存到 localStorage，下次打开页面自动恢复。

### 1.4 CSS 变量注入

`src/utils/ui/colors.ts` 中的 `setArcoThemeColor` 是核心函数：

```ts
export function setArcoThemeColor(color: string): void {
  const bodyStyle = document.body.style

  // ① 项目自定义变量 —— 用于 var(--theme-color, #5d87ff)
  bodyStyle.setProperty('--theme-color', color)

  // ② Arco Design 主色（hex 格式）
  bodyStyle.setProperty('--color-primary-6', color)

  // ③ Arco Design 主色（RGB 逗号格式）
  //    Arco 菜单选中态用 rgb(var(--primary-6))，需要这种格式
  const rgb = hexToRgb(color)
  bodyStyle.setProperty('--primary-6', `${rgb[0]}, ${rgb[1]}, ${rgb[2]}`)

  // ④ 生成 9 个明暗变体（按钮 hover、selected 等状态用）
  handleArcoThemeColor(color, useSettingStore().isDark)

  // ⑤ 生成 15 个渐淡色（可选，用于自定义场景）
  for (let i = 1; i < 16; i++) {
    bodyStyle.setProperty(`--color-primary-custom-${i}`, colourBlend(color, '#ffffff', i / 16))
  }
}
```

`handleArcoThemeColor` 负责生成明暗变体：

```ts
export function handleArcoThemeColor(theme: string, isDark: boolean = false): void {
  const bodyStyle = document.body.style

  // 9 个变浅（light）或变深（dark）的变体
  for (let i = 1; i <= 9; i++) {
    bodyStyle.setProperty(
      `--color-primary-light-${i}`,
      getLightColor(theme, i / 10, isDark), // 例：#5D87FF → 逐渐变浅
    )
  }

  // 9 个变深的变体
  for (let i = 1; i <= 9; i++) {
    bodyStyle.setProperty(
      `--color-primary-dark-${i}`,
      getDarkColor(theme, i / 10), // 例：#5D87FF → 逐渐变深
    )
  }
}
```

### 1.5 设置面板 UI

`src/components/core/layouts/zhao-settings-panel/index.vue`：

```html
<!-- 7 个色块，点击切换 -->
<div
  v-for="color in mainColors"
  :key="color"
  class="w-7 h-7 rounded-full cursor-pointer"
  :class="settingStore.systemThemeColor === color ? 'ring-2 ring-offset-2' : ''"
  :style="{ background: color }"
  @click="settingStore.setThemeColor(color)"
/>
```

选中效果用 Tailwind 的 `ring-2 ring-offset-2`，ring 颜色通过 CSS 变量 `--tw-ring-color` 动态设置。

### 1.6 CSS 变量如何被消费

项目中有两种方式引用主题色：

**方式 A：项目自定义变量**

```html
<!-- 直接用 var(--theme-color)，带 fallback -->
<div :style="{ background: 'var(--theme-color, #5d87ff)' }"></div>
```

用于侧边栏 logo 背景、聊天按钮、锁屏头像等。

**方式 B：Arco Design 内部变量**

```css
/* Arco 组件内部 CSS */
.arco-menu-light .arco-menu-item.arco-menu-selected {
  color: rgb(var(--primary-6)); /* 菜单选中文字色 */
  background-color: rgb(var(--primary-6)); /* 选中背景（部分场景） */
}

.arco-btn-primary {
  background-color: rgb(var(--primary-6)); /* 主按钮背景 */
}
```

Arco 组件不需要任何修改，只要 `--primary-6` 变了，所有 Arco 组件的颜色自动跟着变。

### 1.7 为什么设在 `document.body` 上

Arco Design 的 CSS 变量定义在 `body` 选择器上：

```css
/* node_modules/@arco-design/web-vue/es/style/index.css */
body {
  --arcoblue-6: 22,93,255;
  --primary-6: var(--arcoblue-6);
  ...
}
```

如果我们在 `document.documentElement`（`<html>`）上设置，会被 `body` 样式表中的定义覆盖。所以必须在 `document.body.style` 上设置，inline style 优先级高于样式表。

## 二、自定义圆角

### 2.1 整体流程

```
用户拖动滑块
  → settings panel 调用 settingStore.setCustomRadius(value)
    → store 更新 reactive 状态（持久化）
    → 在 document.body.style 上设置 --custom-radius
      → main.css 中将 --custom-radius 映射到 Arco 的圆角变量
        → 所有 Arco 组件的圆角自动更新
```

### 2.2 配置

默认值 `src/config/setting.ts`：

```ts
customRadius: '0.75' // 单位 rem（字符串类型）
```

### 2.3 Store 层

```ts
const customRadius = ref(SETTING_DEFAULT_CONFIG.customRadius)

const setCustomRadius = (radius: string) => {
  customRadius.value = radius
  document.body.style.setProperty('--custom-radius', `${radius}rem`)
}
```

### 2.4 CSS 变量映射

关键在 `src/assets/main.css`：

```css
:root {
  --border-radius-small: var(--custom-radius, 0.375rem);
  --border-radius-medium: var(--custom-radius, 0.5rem);
  --border-radius-large: var(--custom-radius, 0.75rem);
}
```

**原理**：Arco Design 所有组件的圆角都用这三个变量：

```css
/* Arco 组件内部 */
.arco-btn {
  border-radius: var(--border-radius-small);
}
.arco-card {
  border-radius: var(--border-radius-medium);
}
.arco-modal {
  border-radius: var(--border-radius-large);
}
.arco-menu-item {
  border-radius: var(--border-radius-small);
}
```

我们在 `:root` 上用 `var(--custom-radius)` 替换了这三个变量的默认值。当 JS 修改 `--custom-radius` 时，三个 Arco 变量同时变化，所有组件圆角联动更新。

### 2.5 设置面板 UI

```html
<a-slider
  :model-value="Number(settingStore.customRadius)"
  :min="0"
  :max="2"
  :step="0.25"
  @change="(v: number) => settingStore.setCustomRadius(String(v))"
/>
```

滑块范围 0~2rem，步长 0.25rem。值显示在右侧：

```html
<span>{{ settingStore.customRadius }}rem</span>
```

## 三、初始化时机

应用启动时在 `src/main.ts` 中调用 `initializeTheme()`（`src/hooks/core/useTheme.ts`）：

```ts
export function initializeTheme() {
  const settingStore = useSettingStore()

  // 1. 应用暗色/亮色主题
  // 2. 设置主题颜色（从 store 读取持久化的值）
  setArcoThemeColor(settingStore.systemThemeColor ?? '#5D87FF')

  // 3. 设置圆角（从 store 读取持久化的值）
  document.body.style.setProperty('--custom-radius', `${settingStore.customRadius}rem`)
}
```

执行顺序：`initStore(app)` → `initializeTheme()` → `initRouter(app)`

Store 初始化在前，所以 `settingStore.systemThemeColor` 已经从 localStorage 恢复了上次保存的值。

## 五、关键概念总结

1. **CSS 变量是响应式的**：修改一个变量，所有引用它的元素立即更新，不需要重新渲染组件
2. **inline style > 样式表**：`element.style.setProperty()` 产生的 inline style 优先级最高
3. **设在 `body` 而非 `html`**：因为 Arco 的变量定义在 `body` 上，必须在同一元素上覆盖
4. **RGB 格式 vs Hex 格式**：Arco 用 `rgb(var(--primary-6))` 需要 `93,135,255` 格式，不是 `#5D87FF`
5. **映射替换**：圆角通过 `var(--custom-radius)` 替换 Arco 的默认值，一处修改全局生效
